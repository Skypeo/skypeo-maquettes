## Registre pressenti

**Deux options nettes, à trancher :**

### Option A — **R3 Sensoriel / Bien-être** (recommandé pour séduire)
- La mousse, la literie, le sommeil, le confort = territoire sensoriel direct.
- Palette : crème ivoire + jaune mousse (rappel photo atelier) + terracotta doux ou vert sauge + gris ardoise.
- Signature : **tranches de mousse** qui respirent (layers décalés loop 8s), plans macro de matière, typo serif élégant + sans humaniste, espaces larges, ambiance boudoir/atelier parisien design.
- Avantage : séduit tout de suite une gérante qui aime le design, différencie de tous les autres magasins de mousse.

### Option B — **R1 Éditorial Artisan angle enseigne peinte**
- S'appuie sur la devanture rouge iconique. Typo peinte à la main dominante, palette bleu enseigne + rouge vif + crème, composition asymétrique type affiche parisienne 1920-60.
- Avantage : ancre fortement « atelier historique du 18e arrondissement ».
- Angle non encore consommé dans `memory/styles-used.md`.
- Attention : risque de ne pas paraître assez « design contemporain » pour la gérante.
