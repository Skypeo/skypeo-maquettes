# Skypeo-maquettes — Mémoire & commandes de déploiement

## Convention générale

**Se placer dans le repo (une fois par session de terminal) :**

```bash
cd /c/Users/timot/skypeo-maquettes
```

**Push d'un projet `<slug>` :**

```bash
rm -f .git/index.lock
git add <slug>/
git commit -m "<slug>: <message>"
git push
```

**URL de déploiement :** `https://skypeo-maquettes.vercel.app/<slug>`

Vercel déploie automatiquement tout push sur `main`. Compter 30–60s avant que la nouvelle version soit en ligne. En cas de cache persistant : `Ctrl + F5` (hard refresh) ou fenêtre privée.

