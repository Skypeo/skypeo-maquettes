# Toutou Rapetou — Salon de toilettage

## Description
Maquette one-page pour Toutou Rapetou, salon de toilettage pour chiens, chats et lapins aux Ponts-de-Cé (49).

## Stack technique
- HTML unique (`index.html`) avec CSS et JS inline
- Google Fonts : Fredoka (headings) + Quicksand (body)
- Images : Unsplash (site original sans images exploitables)
- Logo : CDN du site original (le-de.cdn-website.com)

## Style visuel
- **Style** : Playful Tender (Tendresse Ludique)
- **Palette** : Crème (#FFF8F0), Bleu ciel (#4AACD4), Marron chocolat (#5C3D2E)
- **Element signature** : Séparateurs mousse de savon (ellipses SVG), bulles flottantes animées
- **Texture** : Grain doux feTurbulence teinté bleu/marron

## Sections
1. Header fixe avec nav + hamburger mobile
2. Hero split (texte + image)
3. Salon — services de toilettage
4. Features — 3 cards (douceur, hygiène, sans rdv)
5. Compteurs animés + galerie animaux bulles
6. Formules mensuelles — 2 pricing cards
7. Toilettage à domicile (section sombre)
8. Articles pour animaux — 3 cards
9. Contact — formulaire + infos + map
10. Footer

## Decisions
- Images Unsplash car le site Duda n'expose pas ses images
- Palette extraite du logo (bleu + marron)
- Foam dividers (mousse de savon) comme élément signature unique
